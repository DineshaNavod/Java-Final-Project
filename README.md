# 🎓 Faculty Management System

## 📌 Project Overview

This is a Java + MySQL-based Object Oriented Programming mini project developed for the Faculty of Technology, University of Ruhuna.

The system is designed to manage academic and administrative activities efficiently, including user management, courses, marks, attendance, medical records, notices, and timetables.

---

## ⚙️ Technologies Used

- Java (OOP, Swing / JavaFX)
- MySQL Database
- JDBC (Database Connectivity)
- Git & GitHub (Version Control)

---

## ✨ Features

### 👤 User Management

- Login system for different user roles
- Profile management

### 📚 Academic Management

- Course management
- Timetable management
- Notice board system

### 📊 Assessment System

- Marks entry and updates
- CA calculation
- Grade and GPA/CGPA calculation

### 📅 Attendance System

- Theory and practical attendance tracking
- Attendance percentage calculation
- Medical record handling and approval

---

## 🗄️ Database

The system uses a MySQL relational database including tables such as:

- users
- courses
- marks
- attendance
- medicals
- notices
- timetables

---

## 🚀 How to Run

1. Clone the repository
2. Open the project in a Java IDE (NetBeans / IntelliJ / Eclipse)
3. Import MySQL database
4. Configure database connection in `DBConnection.java`
5. Run the application
